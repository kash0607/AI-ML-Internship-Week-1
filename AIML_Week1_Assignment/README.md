## Objective
Analyze a student performance dataset to understand academic performance patterns, perform data cleaning, create visualizations, and generate business insights.

## Dataset
Students Performance in Exams

Source:
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Tasks Completed

- Dataset inspection
- Missing value analysis
- Duplicate record analysis
- Feature understanding
- Data cleaning
- Descriptive statistics
- Observations

### Task 2: Data Visualization
- Histogram
- Boxplot
- Scatter Plot
- Correlation Heatmap
- Bar Chart

### Task 3: Business Insight Report
- Performance analysis
- Key findings
- Recommendations
- Limitations
- Conclusion

## Key Findings
1. Students who completed test preparation scored approximately 7.6 points higher.
2. Reading and writing scores showed a strong positive correlation.
3. Most students scored between 60 and 80 marks.
4. Academic performance indicators were strongly correlated.
5. Higher parental education levels were associated with slightly higher student performance.
